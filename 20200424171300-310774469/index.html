쿠팡 브랜드 - 홈플래닛 저소음 무선마우스, 블랙

마우스 킬러가 휴대하기 괜찮은 마우스.

제가 정말 마우스를 험하게(?) 사용하거든요.
스트레스 받으면 손목 스냅을 사용해 마우스를 탁탁! 위아래로 치거나, 짐 챙기다가 떨어뜨리기도 하고요.

그래서 부담 없이 사용할 무선 마우스를 찾다가 홈플래닛 제품을 선택했습니다.
이 전에 사용한 홈플래닛 제품 ( 무드 등, 고데기 )는 문제없이 제 기능을 잘 해서 좋았는데요.
무선 마우스 역시 마음에 들어서 자주 손에 들고 있을 것 같네요.


1. 패키지
- 깔끔한 패키지 안에 ’무선 마우스 + USB 커넥터 + AA 건전지 + 안내서’ 구성으로 들어있습니다.
>> 건전지가 아예 마우스 안에 들어있어서 좋더군요. ( 사용 전에 비닐은 벗겨야 합니다. )

- 마우스 바디, 패키지 안에서 이상한 냄새가 안 나서 좋았습니다.
>> 홈플래닛 무선 키보드는 바디에서 이상한 냄새가 났었는데, 마우스는 냄새가 없어서 첫 느낌이 좋았네요.

2. 마우스 ( 크기, 기능, 사용 전 느낌 등 )
- 생김새는 사진 찍으면 잘 나오는, 완전 기본형 마우스입니다.
>> 보통 마우스 바디에 로고가 찍혀있기 마련인데, 아무것도 없는 깔끔한 블랙이라 만족스럽습니다.

- 마우스 길이는 약 9.5cm, 폭은 약 5.8cm 입니다.
>> 휴대용으로 아주 제격인 크기입니다.
>> 집에 있던 나름 가장 작은 마우스와 크기가 비슷하더라고요.

- 마우스는 바디 무게는 매우 가볍습니다. 건전지 빼고 48g 약간 못 미치더군요.
- 기대 안 했는데, 생각보다 마우스 바디가 균형 있고 안정감 있습니다.
- ( 흔들 일은 없겠지만 ) 흔들면 덜그럭 소리가 조금 나네요.

3. 사용감
- 사용 후에 마우스 밑 부분을 열어 USB 커넥터를 넣어야지 OFF 되는 점이 아쉽습니다.
>> 마우스 바디에 노트북과의 연결을 OFF / ON 할 수 있는 버튼이 있으면 좋겠습니다. 무선 마우스를 사용 시 커넥터를 장시간 꽂아두고 마우스에서만 연결을 끊는 경우가 자주 있었는데, 이 제품은 사용 후에 밑 부분을 열어 커넥터를 집어 넣어야 해서 귀찮을 때가 있더라고요.

- 기존과 동일한 컴퓨터 마우스 설정 환경에서, 홈플래닛 마우스 반응속도가 꽤 괜찮네요.
>> 빡빡하다는 후기가 있어서 걱정했는데, 잘 움직입니다. 이 가격에 대비해서 기대했던 수준으로 움직여줍니다.

- 다양한 소재 위에서 사용할 때 인식이 잘 되고 움직임이 버벅대지 않더군요.
>> 가죽 데스크 매트, 노트북 위, 종이, 목재 테이블에서 사용했는데, 부드럽게 돌아갑니다.

- 장시간 사용하는데 손이 불편하지 않았습니다.
>> 손가락이 긴 편인데, 손에 착착 감깁니다. 그립감이 좋네요.

- 마우스 소음은 사람마다 다를 것 같은데, ‘약간 큰 편에 속하는 저소음’ 느낌입니다.
>> 마우스 좌 우측 버튼 클릭 소리보다 휠 소리가 큽니다.

- USB 커넥터 뺐다 꽂았다 했을 때, 마우스 인식이 빠릅니다.
