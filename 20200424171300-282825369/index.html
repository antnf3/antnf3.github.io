평소 일찍 자고 일찍 일어나는 삶을
꿈꾸기만 하는 대학생입나다.
이러다 꿈만 꾸다 인생 다 가겠다 싶어서
수면체크를 위해 샤오미 미밴드를 구매했습니다.

오늘 와서 개봉하고 지금 왼쪽 팔에 끼고 있는데
아주 만족스럽습니다.
기능도 다양하고 디스플레이 화면도 깔끔합니다.

다양한 기능을 생각하면 가격도 저렴하고
정말 괜찮은 상품이라고 생각합니다.
수면체크를 위해 샀지만 생각보다 쓸모가 훨씬 많을 거 같아요!

구성품
미밴드4, 충전단자(충전기는 없습니다. usb연결잭만 있어요), 설명서

기능 정리
1. 운동 체크 가능: 걷기, 달리기, 사이클링 등등
(걸음 수 측정을 똑바로 하는지는 잘 모르겠습니다. GPS를 안 켜서인지 가만히 있었는데 갑자기 95걸음 걸었다고 그러네요.)

2. 음악 듣기: 스마트폰 음악 듣던 걸 멈추거나 다음곡으로 넘어가게 할 수 있습니다

3. 심박수 측정(지속적 측정, 원할 때만 측정 선택 가능)

4. 수면체크: 자고 일어나면 자동으로 체크됩니다!! 심박수감지를 수면체크모드?로 해놓으면 더 정확히 수면체크를 해줍니다!! 너무너무 만족스러워요!! 미밴드 덕분에 이번 일주일 내내 일찍 일어나기 성공했습니다!!

5. 알림 기능: 전화, 메세지, 카톡 등등
(제가 전화 카톡 안 받기로 친구들 사이에서 유명한데 미밴드가 고쳐주길 바라며....)

6. 알람/스톱워치 기능: 알람이나 스톱워치 설정하면 진동이 울립니다. 강한 진동은 아닙니다.
(매일 아침 저를 깨워주기에는 너무 미미한 수준입니다. 참고로 저는 락음악을 1시간 틀어도 안 일어납니다.)

7. 휴대폰 찾기: 이거 신기하더군요!!
미밴드에 휴대폰 찾기를 누르면 휴대폰에서 삐삐 소리가 납니다.
무음으로 해놨을 때 매우 편리할 것 같습니다.

사용방법
1. 먼저 스마트폰으로 스토어에 들어가 '미핏'을 다운 받습니다.
2. 블루투스와 와이파이를 키고 어플을 열어서 계정을 만들면 서로 연결이 됩니다.
3. 스마트폰으로 '프로필' 화면에 들어가서 MI밴드4를 클릭하면 다양한 설정이 나옵니다.
4. 원하시는 설정을 하시면 되고, 어려우시면 도움말도 있으니 읽어보시고 설정하시면 됩니다.
